## About

This project is built using Jekyll and GitBook. It allows me to write articles and blog posts in Markdown format and publish them using GitHub Pages.

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository: `git clone https://github.com/tomashak/jekyll-gitbook-project.git`
2. Install Jekyll: `gem install jekyll`
3. Build the site: `jekyll build`
4. Serve the site locally: `jekyll serve`
5. Open your browser and visit `http://localhost:4000` to view the site.
